## Screenshot Guidelines

- **Format**: WebP format (with JPG/PNG fallback if needed)
- **Dimensions**: Recommended 1920x1080 or 1440x900 (16:9 or 16:10 aspect ratio)
- **Quality**: High quality, optimized for web (aim for < 500KB per image)
- **Content**: Show the best/most representative view of each project
- **Naming**: Use the format `{project-name}-screenshot.webp`

## Tools for Screenshot Capture

- Browser DevTools (full-page screenshots)
- Screenshot API services
- Puppeteer/Playwright for automated screenshots
- Manual capture tools (Snagit, etc.)
