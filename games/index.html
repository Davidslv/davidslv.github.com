Games

Small browser games written as “play it as it was” studies — original code on an HTML canvas, no engine, no dependencies. Playable with the keyboard or a gamepad.

  • Rally

    A Game Boy Color–style pseudo-3D rally racer. Arrows / X to drive, or plug in a gamepad.

  • Tetris

    NES-style falling blocks, mechanics as they were. Arrows + Z/X, or plug in a gamepad.

  • Stealth

    Top-down stealth — slip past vision cones and patrols to the exit across a 20-level campaign. Arrows + Z, or plug in a gamepad.
