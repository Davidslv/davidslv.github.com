Vanilla Roguelike

GitHub

About

Vanilla Roguelike is a roguelike game written in vanilla Ruby, inspired by the original 1980's Rogue game. It features procedurally generated mazes, player movement, and a flexible architecture based on the Entity-Component-System pattern with an event-driven system for logging and debugging.

Motivation

I've always been fascinated by roguelike games and their procedural generation techniques. This project started as a way to understand game development fundamentals, but it evolved into something much more: an educational journey that I wanted to share with others.

Educational Book

Based on my experience building Vanilla Roguelike, I authored a comprehensive book that teaches readers how to build their own roguelike game from scratch. The book covers:

  • Game architecture and design patterns
  • Entity-Component-System implementation
  • Procedural generation algorithms
  • Event-driven architecture
  • Testing strategies for game development
  • Performance optimization techniques

The goal is to make game development accessible to developers who want to learn by building something tangible and engaging. Whether you're new to game development or looking to understand roguelike mechanics, this book provides a practical, hands-on approach to learning.

About

Cashew is a financial management and budgeting application that helps users track expenses, set financial goals, and gain insights into their spending habits.

Motivation

I created Cashew to address the need for a simple, privacy-focused financial tracking tool. Many existing solutions are either too complex or compromise user privacy. Cashew aims to provide powerful financial insights while keeping user data secure and private.
